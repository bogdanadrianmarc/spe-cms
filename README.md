# Software Product Engineering
Main repository for SPE CMS Project.

# Installing
```shell
$ cd SPE_cms_frontend && npm install
```

# Running locally
```shell
$ cd SPE_cms && mvn spring-boot:run
```

After executing, open up index.html in `/SPE_cms_frontend` for the demo.
